One Place — Quick Walkthrough

This guide shows step-by-step how to demo the prototype to parents or schools.

1) Open the child view
- File: `oneplace-prototype/index.html`
- What the child sees: Today summary with Now / Next / Later, Help Me button.

2) Open the parent dashboard
- File: `oneplace-prototype/parent-dashboard.html`
- Actions: Add Activity, Record Occurrence, Save Child Profile, Preview Support Passport.

3) Add an activity (live demo)
- In Parent Dashboard: fill `Title`, `Date`, `Time`, `Location` and click **Add Activity**.
- Expected: a confirmation alert.
- Switch to Child view and refresh the page — the activity appears under "Later".

4) Share and export Support Passport
- In Parent Dashboard: toggle "Share with school" on an activity, click **Preview Passport**, then **Download PDF**.
- Expected: A printable window opens with child info, shared activities, and recent occurrences.

5) Demonstrate Help Me
- In Child view: click **Help Me**, choose a feeling — the prototype logs the choice (alert) and parents can see occurrences in Parent Dashboard.

6) Tips for demoing
- Use the Child Profile card on the Parent Dashboard to set the child's name, DOB and strategies before demo.
- Tell schools this is a prototype; data is stored locally in the browser for demo only.
- For production: explain that accounts, secure back-end, and parental consent for sharing will be implemented.

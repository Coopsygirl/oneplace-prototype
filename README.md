One Place — Child "Today" screen prototype

Open `index.html` in a browser to see the simple interactive prototype.

Features in prototype:
- Now card with checklist saved in localStorage
- Next transport card with countdown (editable via localStorage)
- Later events list
- Help Me quick prompt (logs choice locally via alert)

How to run:
1. Open the folder `/workspace/oneplace-prototype` in your file explorer.
2. Double-click `index.html` or open it in your browser.
